Ejercicio #3

Escribir una función que permita saber si un número se repite dentro de un arreglo.

Blockquote Casos de prueba:

  1. Caso 1: Resultado esperado: El número 5 se encuentra más de una vez [1, 22, 5, 17, 10, 5, 40, 5]
  2. Caso 2: Resultado esperado: El número 7 se encuentra más de una vez [7, 41, 5, 7, 10, 13, 2]
  3. Caso 3: Resultado esperado: No hay números repetidos [1, 22, 5, 14, 24, 31, 27, 15, 105]
  4. Caso 4: Resultado esperado: No hay números repetidos [50, 11, 12, 14, 24, 31, 27, 15, 105]
